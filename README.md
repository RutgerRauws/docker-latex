# Docker Latex Image

## Packages

a docker image based on **ubuntu 14.04** containing:

* `texlive-full`
* `python-pygments`
* `gnuplot`
* `biber`
* `pandoc`
* `pandoc-citeproc`

... and extra tools:

* `wget`
* `wput`
* `curl`
* `zip`
* `tar`
* `rsync`
* `make`

Inherited from `lang/latex`.

## Install

```sh
docker pull janstuemmel/latex
```
